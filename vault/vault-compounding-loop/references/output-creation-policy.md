# Output Creation Policy

Create an `outputs/` note only when all of these are true:

1. The work is grounded in ingested or already-source-backed vault material.
2. The synthesis is durable and likely to be reused.
3. The note can link back to real extracts, wiki notes, raw captures, or other valid vault provenance.
4. The note improves future retrieval more than it adds clutter.

Do not save ordinary chat assessments, unsupported opinions, or background process summaries just because they were useful in the moment.
