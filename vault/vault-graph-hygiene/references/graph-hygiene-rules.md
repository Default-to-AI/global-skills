# Vault Graph Hygiene Rules

- Prefer durable wiki targets over raw-source targets when both exist.
- Use the first meaningful body mention for a link, not every repetition.
- Broken example wikilinks in prose should usually become code or plain text, not forced vault links.
- Duplicate handling is audit-first, merge-second.
- Orphan notes are a signal to inspect context, not a license for speculative link creation.
- After graph edits, rerun the audit-fix flow.
