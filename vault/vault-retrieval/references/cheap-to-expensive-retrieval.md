# Cheap-to-Expensive Vault Retrieval

1. Open `vault-index.md` to choose the correct domain.
2. Open the domain's `wiki/index.md`.
3. Check `outputs/` for prior source-backed synthesis on the same topic.
4. Search titles, frontmatter, and index entries.
5. Read only the top candidate wiki notes in full.
6. Touch raw captures only when provenance, conflict resolution, or missing detail requires it.

This order keeps retrieval fast, grounded, and aligned with the vault's curated graph rather than blind full-vault reading.
