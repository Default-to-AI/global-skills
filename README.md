# Global-Skills

Hermes agent **skills repository** for the `default` profile, backed up to GitHub.

This repo is the source of truth for agent skills loaded at
`%LOCALAPPDATA%\hermes\skills` (a junction points there). It is mirrored to
`Default-to-AI/global-skills` so skills survive disk loss.

## What's here

- `skills/` — the full Hermes skill library (coding, research, design, vault,
  gbrain, etc.).
- `gbrain-sync-git-retry-patch/` — local patch kit that makes gbrain's
  `sync.ts` `git()` helper retry transient MSYS `spawnSync` timeouts, so the
  `<head>` sentinel stops wedging sync on Windows. Re-apply after any
  `gbrain upgrade` (the running patch lives uncommitted in gbrain's working
  tree).
- `retrieval-reflex/` — gbrain retrieval-reflex policy skill + resolver row
  (installed via `gbrain integrations install retrieval-reflex`).
- `setup-gbrain/`, `sync-gbrain/` — gbrain integration skills for the host
  agent.

## `gstack` gitlink

`gstack/` is a **gitlink (submodule pointer)** to
[`garrytan/gstack`](https://github.com/garrytan/gstack) — a third-party repo
that is **not owned here**. Pull upstream to update it; do not edit or
duplicate it. Its `llms.txt` is generated by that repo's own tooling.

## Notes

- All commits are local-first; this GitHub remote is a backup mirror.
- The `gbrain-sync-git-retry-patch` skill is the durable record of the
  gbrain sync fix; the patched `sync.ts` in the gbrain install is not
  version-controlled by this repo.
